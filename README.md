

# PatchFlow

Stop regenerating entire images just to edit a small detail.

PatchFlow is a Chrome extension that brings precise AI image editing directly into ChatGPT, Gemini, and other web-based AI interfaces.

Instead of sending an entire image to the AI and risking unwanted changes, PatchFlow lets you crop a specific region, edit only that portion with AI, and automatically merge the result back into the original image at full resolution.

AI image editors often recreate large parts of an image, causing unwanted changes to faces, text, backgrounds, logos, and other details you never intended to modify.

PatchFlow solves this by limiting the AI's scope to only the area you select.

### Benefits

* Edit only the exact region you want
* Preserve the rest of the image perfectly
* No API keys required
* No developer setup
* Works directly inside ChatGPT and Gemini
* Uses AI services you're already using
* Full-resolution output
* Fast drag-and-drop workflow

### How it Works

1. Select an image.
2. Crop the area you want to edit.
3. Paste the crop into ChatGPT or Gemini with your prompt.
4. Drag the AI-generated result back into PatchFlow.
5. PatchFlow automatically merges it into the original image and downloads the final version.

No APIs. No billing setup. No complex tools.
